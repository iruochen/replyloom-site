# Publish the ReplyLoom website with GitHub Pages

The Chrome Web Store needs a stable public privacy-policy URL. The extension source code does not need to be public.

The website is bilingual and includes the product demo video. The public repository is `iruochen/replyloom-site`.

Deployment:

1. Copy the complete contents of this folder into the public `replyloom-site` repository root.
2. In the repository, open **Settings → Pages**.
3. Under **Build and deployment**, choose **Deploy from a branch**.
4. Select the `main` branch and `/ (root)`, then save.
5. Wait for GitHub Pages to publish the site.

Published URLs:

- Website: `https://iruochen.github.io/replyloom-site/`
- Privacy policy: `https://iruochen.github.io/replyloom-site/privacy.html`

Open that URL in a private/incognito browser window before entering it in the Chrome Web Store dashboard.
